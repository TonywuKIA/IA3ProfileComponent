# IA3 – Profile Header with Overlay Card

## Overview
This project implements a profile header screen using Jetpack Compose and Material 3.

The layout demonstrates Box layering with a background header, a circular avatar, and a Card that partially overlaps the header to create a classic profile card effect.

## Layout Techniques Used
- Box layering for background and foreground
- contentAlignment and align for positioning
- offset for overlap effect
- zIndex to control layering order
- clip(CircleShape) for avatar
- aspectRatio for header sizing
- Card elevation for shadow effect

## Material 3 Components
- TopAppBar
- Card
- Button
- FilledTonalButton
- AssistChip
- Icon
- Surface
